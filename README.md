# Learn by Ewonz 📘

A study paper sharing app for learners around the world 🌍

## 🌏 Supported Languages & Focus
- 🇯🇵 Japanese (Main Focus)
- 🇱🇰 Sinhala (Main Focus)
- 🇨🇳 Chinese
- 🇰🇷 Korean
- 🇷🇺 Russian
- And all available study languages in Sri Lanka (English, Tamil, etc.)

## 🚀 Features
- 📤 Upload PDF papers & notes
- 📄 View papers online
- ⬇️ Download study materials
- 🔍 Search papers easily
- 📚 Category-based learning system

## 🎯 Purpose
To help students access and share study materials easily, especially for Japanese and Sinhala learners, while supporting multiple global languages.

## 🛠 Built With
- Flutter (Mobile App)
- Firebase (Backend & Storage)

## 💡 Vision
"Learn by Ewonz" is a simple, modern study hub for sharing knowledge across languages and countries 🌍
